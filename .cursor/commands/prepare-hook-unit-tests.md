Методика написания тестов:
1. Используй метод `renderHook` для рендера и метод `act` для взаимодействия.
Пример теста:
    ```ts
    import { renderHook, act } from '@testing-library/react';
    import { useCounter } from '../hooks/useCounter';
    describe('useCounter', () => {
      it('should increment counter', () => {
        const { result } = renderHook(() => useCounter());
        act(() => {
          result.current.increment()
        });
        expect(result.current.count).toBe(1);
      });
    });
    ```

2. Если тебя просят сделать мок для какой-то сущности, делай. А если не знаешь как сделать то просто делай заготовку и проси разработчика самому заполнить мок:
    ```ts
    vi.mock('some-dependency-name', async () => {
      // define your mock here
      return {};
    });
    ```
3. Файл с тестами на хук должен иметь название `hookName.spec.ts(x)`. Даже если хук лежит в файле `hooks.ts(x)`, создавай отдельный файл с тестами для каждого из хуков.
