# Popover

## Installation
`npm i @snack-uikit/popover`

[Changelog](./CHANGELOG.md)

## Description

- Пакет `@snack-uikit/popover` предоставляет компонент `Popover` для отображения всплывающих подсказок и дополнительной информации рядом с элементом-триггером.
- Компонент поддерживает различные способы активации: по клику, наведению, фокусу или их комбинациям, что позволяет адаптировать поведение под разные сценарии использования.
- Поповер может быть позиционирован относительно триггера в 12 различных вариантах (сверху, снизу, слева, справа с вариантами start/end) и автоматически корректирует своё положение при нехватке места на экране.
- Компонент работает в контролируемом и неконтролируемом режимах, поддерживает задержки открытия/закрытия при наведении, управление закрытием по клику вне области и закрытие при навигации по истории браузера.
- Триггер поповера может быть передан как дочерний элемент (children) или через ref на внешний элемент, что даёт гибкость в структуре компонентов.
- Поповер отображается с указательной стрелкой, направленной на триггер, и имеет ограничение максимальной ширины по умолчанию (которое можно отключить через `disableMaxWidth`).
- Компонент `Popover` использует под собой компонент `PopoverPrivate` для базовой функциональности позиционирования и управления состоянием.
- Figma: [`Popover`](https://www.figma.com/file/jtGxAPvFJOMir7V0eQFukN/Snack-UI-Kit-1.1.0?node-id=2%3A10&mode=design).

## Example

```tsx
import { Popover } from '@snack-uikit/popover';
import { ButtonFilled } from '@snack-uikit/button';
import { Link } from '@snack-uikit/link';

function Example() {
  return (
    <Popover
      tip={
        <div>
          Не нажимайте эту кнопку, пожалуйста
          <br />
          <Link href='#' text='узнать почему' />
        </div>
      }
      trigger='click'
      placement='top'
    >
      <ButtonFilled label='Кнопка с подсказкой' />
    </Popover>
  );
}
```
