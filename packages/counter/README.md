# Counter

## Installation
`npm i @snack-uikit/counter`

[Changelog](./CHANGELOG.md)

## Example

```typescript jsx
import { Counter } from "@snack-uikit/counter";

<Counter
  value={42}
  size='m'
/>

<Counter 
  value={101}
  appearance='critical'
  variant='count-plus'
  plusLimit={100}
/>
```
