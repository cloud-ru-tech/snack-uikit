# Counter

## Installation
`npm i @snack-uikit/counter`

[Changelog](./CHANGELOG.md)

## Example

```typescript jsx
import {Counter} from "@snack-uikit/counter";

<Counter
  value={42}
  size={Counter.sizes.M}
/>

<Counter 
  value={101}
  appearance={Counter.appearances.Critical}
  variant={Counter.variants.CountPlus}
  plusLimit={100}
/>
```
