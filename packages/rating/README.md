# Rating

## Installation
`npm i @snack-uikit/rating`

[Changelog](./CHANGELOG.md)

## Description

- Компонент `Rating` позволяет пользователям оценивать контент, товары или услуги с помощью визуального представления в виде звёзд.
- Поддерживает **контролируемый и неконтролируемый режимы** работы через пропы `value` и `defaultValue`.
- Позволяет настраивать **количество звёзд** (`count`, по умолчанию 5) и **начальное значение рейтинга** (`defaultValue`).
- Поддерживает **половинчатые звёзды** (`allowHalf`) для более точной оценки.
- Может работать в **режиме только чтения** (`readonly`), когда пользователь не может изменить рейтинг.
- Поддерживает **сброс рейтинга** при повторном клике на выбранную звезду (`allowClear`).
- Предоставляет **различные цветовые схемы** (`appearance`: primary, red, orange, yellow, green, blue, violet, pink) для визуального оформления звёзд.
- Поддерживает **клавиатурную навигацию** (Enter/Space) для доступности и удобства использования.
- Figma: [`Rating`](https://www.figma.com/design/jtGxAPvFJOMir7V0eQFukN/Snack-UI-Kit-5.0.0?node-id=6638-21541).

## Example

```tsx
import { Rating } from '@snack-uikit/rating';

function Example() {
  return (
    <Rating
      count={5}
      defaultValue={0}
      allowHalf={false}
      allowClear={true}
      appearance="yellow"
      onChange={(value) => console.log('Rating changed:', value)}
    />
  );
}
```
