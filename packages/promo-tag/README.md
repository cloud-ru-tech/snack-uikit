# Promo Tag

## Installation
`npm i @snack-uikit/promo-tag`

[Changelog](./CHANGELOG.md)

## Description

- Пакет `@snack-uikit/promo-tag` предоставляет компактный промо‑тег (`PromoTag`) для акцентирования коротких меток, статусов или промо‑сообщений в составе других интерфейсных блоков.
- Компонент поддерживает несколько визуальных стилей через `appearance` (нейтральный, цветные варианты и т. д.) и семантические цвета `color` (accent/decor), что позволяет согласовать тег с окружающим контентом или брендингом.
- Размер тега (`size`) влияет не только на габариты, но и на доступность дополнительных узлов: для больших размеров можно добавить иконки или произвольные элементы до (`beforeContent`) и после (`afterContent`) текста.
- При передаче обработчика `onClick` тег становится интерактивным: получает роль кнопки и `tabIndex`, что позволяет использовать его как кликабельный бейдж/триггер для действий.
- Текст внутри тега задаётся строкой `text` и отображается в подходящем по кеглю варианте типографики `Typography.SansLabelS` или `Typography.SansLabelM` в зависимости от размера.
- Figma: [`Promo Tag`](https://www.figma.com/file/jtGxAPvFJOMir7V0eQFukN/Snack-UI-Kit-1.1.0?node-id=41%3A27190&mode=design).

## Example

```tsx
import { PromoTag } from '@snack-uikit/promo-tag';
import { PlaceholderSVG } from '@snack-uikit/icons';

export function PromoTagExample() {
  return (
    <PromoTag
      text="Скидка 20%"
      appearance="primary"
      color="accent"
      size="xs"
      beforeContent={<PlaceholderSVG size={16} />}
      afterContent={<PlaceholderSVG size={16} />}
      onClick={() => console.log('PromoTag clicked')}
      data-test-id="promo-tag"
    />
  );
}
```
