# Timeline

## Installation
`npm i @snack-uikit/timeline`

[Changelog](./CHANGELOG.md)

## Description

- Пакет `@snack-uikit/timeline` предоставляет компоненты для отображения временной шкалы событий: основной контейнер `Timeline` для построения полного таймлайна из массива элементов и отдельный элемент `TrackItem` для самостоятельного использования.
- Компоненты поддерживают различные варианты отображения контента (слева, справа, чередование), настройку внешнего вида точек (варианты и цветовые схемы), стили линий (сплошная, пунктирная) и возможность отображения дополнительного контента напротив основного.
- Таймлайн автоматически определяет роли элементов (начало, середина, конец) и управляет отображением соединительных линий между точками.

## Timeline

### Description

- `Timeline` — основной компонент для построения временной шкалы событий из массива элементов.
- Компонент принимает массив элементов (`items`) и автоматически определяет их роли (начало, середина, конец) для корректного отображения соединительных линий.
- Поддерживает единое положение контента для всех элементов (`contentPosition: "right" | "left"`) или чередующееся расположение (`alternate`), когда элементы автоматически размещаются по разные стороны от линии таймлайна.
- Позволяет растянуть таймлайн на всю ширину контейнера (`fullWidth`) и настроить внешний вид каждого элемента индивидуально (вариант точки, цветовая схема, стиль линии, дополнительный контент напротив).
- Figma: [`Timeline`](https://www.figma.com/file/jtGxAPvFJOMir7V0eQFukN/Snack-UI-Kit-1.1.0?node-id=41%3A30522&mode=design).

### Example

```tsx
import { Timeline } from '@snack-uikit/timeline';

function Example() {
  return (
    <Timeline
      items={[
        {
          content: <div>Начало процесса</div>,
          dotAppearance: 'primary',
        },
        {
          content: <div>Промежуточный этап</div>,
          lineStyle: 'dashed',
        },
        {
          content: <div>Завершение</div>,
        },
      ]}
      contentPosition="right"
      alternate={false}
    />
  );
}
```

## TimelineItem

### Description

- Компонент, описанный в сторис-файле `TimelineItem.story.tsx`, в реальном коде используется как `TrackItem`.
- `TrackItem` — отдельный элемент таймлайна, который можно использовать самостоятельно для построения кастомных временных шкал или в составе компонента `Timeline`.
- Элемент состоит из точки (`dotVariant`, `dotAppearance`), соединительных линий (`lineStyle`, `showLines`) и контента, который может располагаться слева или справа от линии (`contentPosition`).
- Поддерживает отображение дополнительного контента напротив основного (`opposite`) и автоматическое определение роли элемента (`role: "start" | "center" | "end"`) для корректного отображения линий.
- Figma: [`TimelineItem`](https://www.figma.com/file/jtGxAPvFJOMir7V0eQFukN/Snack-UI-Kit-1.1.0?node-id=41%3A30522&mode=design).

### Example

```tsx
import { TrackItem } from '@snack-uikit/timeline/src/helperComponents/TrackItem';

function Example() {
  return (
    <TrackItem
      content={<div>Содержимое элемента</div>}
      contentPosition={TrackItem.contentPositions.Right}
      role={TrackItem.roles.Start}
      lineStyle={TrackItem.lineStyles.Default}
      dotVariant={TrackItem.dotVariants.Default}
      dotAppearance={TrackItem.dotAppearances.Primary}
      opposite={<div>Дополнительный контент</div>}
    />
  );
}
```
