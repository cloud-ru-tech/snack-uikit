# Breadcrumbs

## Installation
`npm i @snack-uikit/breadcrumbs`

[Changelog](./CHANGELOG.md)

## Description

- Пакет `@snack-uikit/breadcrumbs` предоставляет компонент навигации «хлебные крошки» (`Breadcrumbs`) для отображения пути пользователя внутри разделов приложения.
- Компонент принимает массив элементов `items` с идентификатором, основным лейблом и опциональными полями (`shortLabel`, `href`, `onClick`, `icon`) и автоматически управляет их отображением в зависимости от доступной ширины контейнера.
- При недостатке места цепочка крошек умно сворачивается: сохраняются крайние элементы, промежуточные могут заменяться короткими подписями или коллапсом, что помогает поддерживать читаемость навигации на разных разрешениях.
- Поддерживаются размеры `size="xs" | "s"`, настройка текстового разделителя (`separator`), отображение только иконки для первого элемента (`firstItemIconOnly`) и принудительное отключение клика у последнего элемента (`inactiveLastItem`).
- Проп `lastEmpty` позволяет зарезервировать место под последний элемент, когда его лейбл ещё неизвестен (например, при асинхронной загрузке данных).
- Figma: [`Breadcrumbs`](https://www.figma.com/file/jtGxAPvFJOMir7V0eQFukN/Snack-UI-Kit-1.1.0?node-id=41%3A30976&mode=design).

## Example

```tsx
import { Breadcrumbs } from '@snack-uikit/breadcrumbs';
import { HomeSVG } from '@snack-uikit/icons';

const items = [
  {
    id: 'home',
    label: 'Главная',
    shortLabel: 'Дом',
    icon: HomeSVG,
    href: '/',
  },
  {
    id: 'section',
    label: 'Раздел',
    href: '/section',
  },
  {
    id: 'subsection',
    label: 'Подраздел',
    shortLabel: 'Подраздел',
    href: '/section/subsection',
  },
  {
    id: 'current',
    label: 'Текущая страница',
  },
];

export function BreadcrumbsExample() {
  return (
    <Breadcrumbs
      items={items}
      size='s'
      separator='›'
      firstItemIconOnly
      inactiveLastItem
      data-test-id='breadcrumbs'
    />
  );
}
```

Массив объектов, описывающих компонент:
```ts
type Item = {
  /* Уникальный id */
  id: string,
  /* Текст */
  label: string;
  /* Короткий вариант текста */
  shortLabel?: string;
  /* Ссылка */
  href?: string;
  /* Колбек обработки клика в айтем */
  onClick?(): void;
}
```
Компоненты Breadcrumbs мемоизирован, старайтесь передавать `items` одной и той же ссылкой.
