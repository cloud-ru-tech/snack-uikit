# Typography

## Installation
`npm i @sbercloud/uikit-product-typography`

## Usage

На данный момент из пакета предоставляются только стили шрифтов без привязки к какому-либо тегу.

Актуальный список linaria-css экспортов:
```text
H1_STYLES 
H2_STYLES
H2_SEMIBOLD_STYLES
H3_STYLES
H3_SEMIBOLD_STYLES
H4_STYLES
H4_SEMIBOLD_STYLES
H5_STYLES
TEXT_1_STYLES
TEXT_2_STYLES
TEXT_3_STYLES
TEXT_4_STYLES
TABLE_TEXT_STYLES
NOTIFY_STYLES
MONO_14_STYLES
MONO_14_BOLD_STYLES
CHARTS_GRID_STYLES
```
