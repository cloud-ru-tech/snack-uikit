# Carousel

## Installation

`npm i @snack-uikit/carousel`

[Changelog](./CHANGELOG.md)

## Description

- Пакет `@snack-uikit/carousel` предоставляет компонент `Carousel` для создания каруселей и слайдеров с поддержкой навигации по элементам.
- Компонент позволяет отображать несколько элементов одновременно, настраивать количество видимых элементов и шаг прокрутки, поддерживает свайп-жесты для мобильных устройств и автоматическую прокрутку слайдов.
- Карусель включает встроенные элементы управления: стрелки навигации (prev/next) и пагинацию для переключения между страницами, которые можно включать или отключать по необходимости.
- Поддерживает цикличную прокрутку (`infiniteScroll`) для бесконечного переключения слайдов, контролируемое и неконтролируемое состояние через проп `state`, а также настройку видимости стрелок управления (по ховеру или всегда).
- Компонент автоматически рассчитывает размеры элементов на основе контейнера и количества отображаемых элементов, адаптируется к изменению размеров окна и скрывает невидимые элементы из DOM для оптимизации производительности.
- Figma: [`Carousel`](https://www.figma.com/file/jtGxAPvFJOMir7V0eQFukN/Snack-UI-Kit-1.1.0?node-id=44%3A7537&mode=design).

## Example

```tsx
import { Carousel } from '@snack-uikit/carousel';
import { Card } from '@snack-uikit/card';

function Example() {
  const items = Array.from({ length: 12 }).map((_, i) => (
    <Card key={i} header={<Card.Header title={`Item ${i + 1}`} />}>
      Content for item {i + 1}
    </Card>
  ));

  return (
    <Carousel
      showItems={2.5}
      scrollBy={2}
      transition={0.4}
      swipe={true}
      arrows={true}
      pagination={true}
      infiniteScroll={true}
      autoSwipe={5}
      controlsVisibility="hover"
      gap="var(--dimension-2m)"
    >
      {items}
    </Carousel>
  );
}
```
